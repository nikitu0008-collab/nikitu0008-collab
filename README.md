### 🔹 `CppCraft` – Code scaffolding for C++ backend services
[Repository →](https://github.com/nikitu0008-collab/CppCraft)

- Provides project templates with CMake, logging, config loading, and basic server skeleton.
- Designed to reduce boilerplate for new C++ backend services.
- Includes examples of multithreaded worker pools and signal handling.

### 🔹 `College Messenger` – Async chat server (work in progress)
*Stack: C++20, Boost.Asio, PostgreSQL*

- Asynchronous TCP server handling 1000+ concurrent clients.
- Features: authentication, persistent sessions, chat rooms, message history.
- Uses PostgreSQL for user data and message storage with connection pool.
- Performance optimized with `io_context` thread pool and zero‑copy message passing.

🚧 *Status: implementing room management and message broadcasting – expected MVP Q3 2026.*
