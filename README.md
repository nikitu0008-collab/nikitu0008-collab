[GitHub →](https://github.com/nikitu0008-collab/CppCraft)

Генерирует скелет сервера: CMake + logging (spdlog) + конфиги + пул потоков. Экономит время на старте.

### College Messenger — асинхронный чат (в разработке)
*C++20, Boost.Asio, PostgreSQL*

Асинхронный TCP‑сервер (io_context + пул потоков).  
Реализовано: авторизация, комнаты, история сообщений в БД.  
Планируется: поддержка 1000+ клиентов.
